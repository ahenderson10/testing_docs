# Getting Started

This section walks you from an empty machine to a quantized model running on the VectorBlox simulator in three steps:

1. [Prerequisites]({% link getting-started/prerequisites.md %}) — operating system, `git-lfs`, and host driver notes.
2. [Install the SDK]({% link getting-started/install.md %}) — clone the repository, install system dependencies, and activate the Python virtual environment.
3. [Run Your First Tutorial]({% link getting-started/first-tutorial.md %}) — pick a model from `$VBX_SDK/tutorials/`, run its shell script, and confirm the simulator output.

Once a tutorial produces a `.vnnx` binary, continue to [Tutorials]({% link tutorials/index.md %}) for the full yolov5n walkthrough and to [Hardware]({% link hardware/index.md %}) to deploy on a PolarFire SoC Video Kit.
